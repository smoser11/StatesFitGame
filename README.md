# StatesFitGame

A React Native mobile game built with Expo where players determine if one U.S. state can fit inside another by rotating and comparing their geographic shapes.

## Play Online

The game is deployed and working at: **https://smoser11.github.io/StatesFitGame/**

## Development

```bash
# Start development server
npm start

# Run on specific platforms
npm run android
npm run ios
npm run web

# Build and deploy to GitHub Pages
npm run deploy
```

## Tech Stack

- React Native with Expo SDK 52
- TypeScript
- @turf/turf for geometric calculations
- React Navigation
- Webpack for web builds (configured for GitHub Pages compatibility)
